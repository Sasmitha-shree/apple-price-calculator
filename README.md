# Java Apple Price Calculator

This is a beginner-friendly Java program that calculates the total amount to be paid for apples based on the price of one apple and the number of apples purchased.

## Features

* Takes user input using `Scanner`
* Uses methods to perform calculations
* Demonstrates parameter passing in Java
* Displays the total amount

## Technologies Used

* Java
* Scanner Class

## Example

### Input

```
Enter the apple price :
20
Enter the apple count : 
5
```

### Output

```
Total price = 100
```

## How to Run

1. Clone the repository.
2. Compile the program:

   ```bash
   javac garden.java
   ```
3. Run the program:

   ```bash
   java garden
   ```
4. Enter the apple price and quantity when prompted.

## Learning Concepts

* Java Classes and Objects
* Methods
* Parameters
* User Input with `Scanner`
* Basic Arithmetic Operations
